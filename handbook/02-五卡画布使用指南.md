# 第二章：五卡画布使用指南

## 画布概览

五卡画布是一张A4纸。布局很简单：

```
        [决策卡]       [方法卡]
             ↘         ↙
              [案例卡]
             ↙         ↘
        [风险卡]       [偏好卡]
```

- **案例卡在正中间** — 它是锚点，其他四张卡都从它派生
- **决策卡和方法卡在上方** — 向上提炼：从经验到逻辑
- **风险卡和偏好卡在下方** — 向下沉淀：从经验到教训与品味

> 画布打印版：[下载五卡画布](../canvas/five-card-canvas.html)

---

## 核心原则：从中间开始

**永远从案例卡开始。** 不要先想"我的方法论是什么"——那是抽象问题，会让你卡住。

案例卡问一个具体问题：*"最近发生了一件什么事，让你觉得自己做对了（或做错了）？"*

具体的故事是你的入口。从一个案例出发，其他四张卡自然就会"长出来"。

---

## 填写顺序

### 第一步：案例卡（5分钟）

用 3-5 句话描述一个具体事件：

- 时间、地点、人物
- 发生了什么
- 你的角色和行动
- 结果如何

> **例子**："2024年6月，公司面临一个关键客户续约谈判。客户要求降价30%。我是项目负责人。我没有直接还价，而是先花了两周做了客户业务分析，发现他们有一个我们没解决的需求。最后以原价续约，并且签了更大的合同。"

### 第二步：决策卡（5分钟）

从这个案例中提取 1-3 个关键决策点：

- 你当时面临什么选择？（至少列出你排除了的选项）
- 你选了哪个？依据是什么？
- 现在回头看，这个决策对吗？

> **从上面的例子提取**：决策点1 — "客户要降价 vs 我们找新价值点 → 选了后者，因为降价是不可逆的恶性循环"

### 第三步：方法卡 + 风险卡（10分钟）

**方法卡**：这个案例中，有没有什么做法你下次还会用？如果同样的做法你在三个以上的案例中都用了，它值得成为一张方法卡。

> **从上面的例子提取**："三不还价法：①先做客户业务分析 ②找到未说出的需求 ③用新价值替代价格谈判"

**风险卡**：这个案例中有什么差点出错的地方？或者同类场景中你见过哪些失败模式？

> **从上面的例子提取**："续约谈判中最危险的信号：客户只谈价格不谈价值——这意味着他们已经找了竞品比价"

### 第四步：偏好卡（5分钟）

这不是案例直接"推导"出来的，而是反思：通过这个案例，我看到了自己什么样的偏好或风格？

> **从上面的例子提取**："我不喜欢对抗性谈判。我偏好把零和博弈转化为双赢探索。"

## 物理准备

| 物品 | 用途 |
|:---|:---|
| 打印的五卡画布（A4） | 主工作区 |
| 便利贴（3种颜色） | 案例卡=黄色，决策卡=蓝色，方法卡=绿色，风险卡=红色，偏好卡=粉色 |
| 笔（2支不同颜色） | 黑色写内容，红色标记关联线和疑问 |
| 计时器 | 控制每张卡的时间 |

如果你用平板+手写笔，可以直接在PDF/图片版画布上书写。

---

## 时间指南

| 阶段 | 时间 | 说明 |
|:---|:---|:---|
| 第一次填充（完整流程） | **25分钟** | 选一个中等复杂度的案例 |
| 后续填充（单案例） | **15分钟** | 熟练后大幅加速 |
| 回顾与连线 | **10分钟** | 跨案例找模式，发现方法卡 |

> **关键规则**：每张卡限时写，不要追求完美。第一版是"够用就好"。

## 常见新手错误

### 错误 1：从方法卡开始
"我先总结一下我的方法论" → 你会写出一堆正确的废话。**方法必须从案例中长出来。**

### 错误 2：案例太模糊
"有一次我处理了一个很难的项目" → 这不是案例。案例需要具体的时间、人名（可用代号）、数字、结果。**越具体，越有用。**

### 错误 3：跳过风险卡
"这个案例好像没什么风险" → 每个案例都有风险。至少问自己：什么差点出错？如果我当时不在场会怎样？

### 错误 4：一张卡写太多
每张便利贴只能写一个点。如果案例卡写了8个要点——那是8个案例。拆开。

### 错误 5：追求一次性填完美
五卡画布是活的文档。第一次填30%就够了。下次回顾同一个案例时，你会看到当时看不见的东西。

## 你的第一张画布：检查清单

- [ ] 打印（或打开）五卡画布：[下载画布](../canvas/five-card-canvas.html)
- [ ] 准备便利贴 + 笔 + 计时器
- [ ] 选定一个案例：**最近3个月内，一件你印象最深的工作事件**
- [ ] 设25分钟计时器
- [ ] 案例卡 →（5分钟）
- [ ] 决策卡 →（5分钟）
- [ ] 方法卡 + 风险卡 →（10分钟）
- [ ] 偏好卡 →（5分钟）
- [ ] 拍一张照片留存
- [ ] 把卡片内容转录到这里（或直接贴在画布上存档）

## 从一张画布到五卡体系

第一张画布完成后，你会得到 **1个案例 + 1-3个决策 + 1-2个方法 + 1-2个风险 + 1个偏好**。

当你完成 5-10 张画布后，跨案例的模式会自然浮现：

- 同一种风险出现在多个案例中 → 这是一条"定律级"风险卡
- 同一种方法在三个以上案例中验证 → 这是一张经过验证的方法卡
- 你的偏好在不同案例中保持一致 → 这是你的认知指纹

这就是五卡体系从"单次练习"到"知识资产"的跃迁过程。
